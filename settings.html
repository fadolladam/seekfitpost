How to set Environment Variables in Vercel

Go to your Vercel project → Settings → Environment Variables → Add each variable below.

OpenAI

Powers AI job post generation

Variable to set:

OPENAI_API_KEY = sk-proj-...
  1. Go to platform.openai.com/api-keys
  2. Click "Create new secret key"
  3. Copy the key and paste it as OPENAI_API_KEY in Vercel

Unsplash

Free professional images for your job posts

Variable to set:

UNSPLASH_ACCESS_KEY = your-access-key
  1. Go to unsplash.com/developers → Your apps → New Application
  2. Accept the API guidelines (require photo attribution — already handled in the UI)
  3. Copy the Access Key (not the Secret Key) and set it as UNSPLASH_ACCESS_KEY in Vercel
  4. Free tier: 50 requests/hour — more than enough for this use case

Telegram

Post to your channel & group

Variables to set:

TELEGRAM_BOT_TOKEN = 123456:ABC-DEF...
TELEGRAM_CHANNEL_ID = @YourChannelName or -100123456789
TELEGRAM_GROUP_ID = @YourGroupName or -100987654321
  1. Open Telegram → search @BotFather → send /newbot
  2. Follow the prompts → BotFather gives you a bot token → set as TELEGRAM_BOT_TOKEN
  3. Add your bot as an Admin to your channel and group
  4. For channel/group IDs: use the @username (e.g. @SeekFitJobKH) or find the numeric ID using @userinfobot
  5. Set TELEGRAM_CHANNEL_ID and TELEGRAM_GROUP_ID in Vercel

Facebook Page

Auto-post to your Facebook page

Variables to set:

FACEBOOK_PAGE_ID = 123456789012345
FACEBOOK_PAGE_TOKEN = EAABsbCS...
  1. Go to developers.facebook.com → Create App → choose "Business" type
  2. Add the Facebook Login for Business product
  3. Go to Graph API Explorer → select your app → select your Page under "User or Page"
  4. Add permissions: pages_manage_posts, pages_read_engagement
  5. Click "Generate Access Token" → copy the token
  6. To get a long-lived token (60 days): call GET /oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=SHORT_TOKEN
  7. Your Page ID is in your Facebook Page URL or in About → Page Transparency

LinkedIn Company Page

Auto-post to your LinkedIn organization · Token expires every 60 days

LinkedIn access tokens expire every 60 days. You will need to repeat this process to refresh the token. LinkedIn is the most complex setup of the four platforms.

Variables to set:

LINKEDIN_ACCESS_TOKEN = AQX...
LINKEDIN_ORGANIZATION_ID = 12345678
  1. Go to linkedin.com/developers → Create App → link it to your Company Page
  2. Under Products, request access to "Share on LinkedIn" and "Marketing Developer Platform"
  3. Under Auth, add redirect URL: https://your-vercel-domain.vercel.app
  4. Add OAuth 2.0 scopes: w_organization_social, r_organization_social
  5. Use the OAuth 2.0 flow to get an access token. You can use LinkedIn OAuth playground or Postman
  6. Your Organization ID is the number in your LinkedIn Company Page URL: linkedin.com/company/12345678
  7. Set both values in Vercel. Repeat every 60 days when the token expires
